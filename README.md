# MSF-D-Control — Регистратура оперативника

## Обзор
Веб-приложение для генерации и регистрации оперативных кодов ЧВК "MSF-043". Предназначено для оформления оперативников с присвоением уникальных идентификаторов MSF и аудиторских номеров.

## Функциональные возможности
### Основной функционал
- **Генерация кода MSF** — создание уникального кода формата `MSF-XXXX-XXXX-XXXX`
- **Генерация аудита MSF** — создание номера формата `MSF-D-XXXXXX-X`
- **Совместная генерация** — одновременное создание обоих кодов
- **QR-код** — автоматическое формирование QR-кода со всеми данными оперативника
- **Регистрация** — копирование QR-кода и текстовых данных в буфер обмена

### Ввод данных оперативника
| Поле | Описание | Требования |
|------|----------|-------------|
| Позывной | Идентификатор оперативника | 2-8 символов, только латиница, без цифр |
| DS-ID | Discord ID оперативника | До 32 символов |
| Корпус | Выбор специализации | SSC / RSC / MSC / TSC |

### Службы (Корпуса)

| Аббр. | Полное название | Назначение |
|-------|----------------|------------|
| SSC | Strike Service Corps | Штурмовой корпус |
| RSC | Recon Service Corps | Разведывательно-снайперский корпус |
| MSC | Medical Service Corps | Медицинский корпус |
| TSC | Technical Service Corps | Технический корпус |

## Архив
- Автоматическое сохранение всех сгенерированных кодов в локальном хранилище браузера
- Фильтрация записей по коду или позывному
- Экспорт архива в JSON-файл
- Копирование кодов из архива при клике

## Интерфейс
- Тёмная тема оформления
- Моноширные шрифты для кодовых значений
- Адаптивная вёрстка под мобильные устройства
- Анимированные переходы и уведомления

## Технологии
- HTML5
- CSS3 (адаптивная вёрстка, CSS Grid, Flexbox)
- JavaScript (ES6+)
- QRCode.js — генерация QR-кодов
